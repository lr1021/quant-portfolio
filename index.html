Projects

  • Option Pricing Numerical Methods

    The purpose of this repository is to provide a practical implementation of various numerical methods for pricing options based on Hull, J. C. (2003). Options, Futures, and Other Derivatives (2nd ed.). Prentice-Hall, specifically Chapter 14: Numerical Procedures. Methods include Binomial Trees, Implicit and Explicit Finite Difference procedures and Monte Carlo Simulation. Beyond option pricing, estimation of hedging parameters is also implemented. Project code is in Python, with Jupyter notebooks to showcase implementation of defined classes and methods.
